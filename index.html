Autonomous MAV Navigation in GPS-Denied Environments

Research Autonomous Systems Lab · ETH Zurich
ROS 2 C++ VIO SLAM PyTorch

Paper GitHub Video

TL;DR: End-to-end pipeline for micro aerial vehicle navigation using visual-inertial odometry and onboard depth estimation. Achieves sub-10 cm drift over 100 m indoor trajectories.

We present a complete navigation stack for GPS-denied indoor flight, combining a stereo visual-inertial odometry front-end with a lightweight onboard depth network for obstacle avoidance. The system runs in real-time on embedded hardware and has been validated on a custom micro aerial vehicle platform across a range of indoor environments.

6-DOF Robotic Arm with Force Feedback

Hardware ETH Zurich
C++ STM32 CAN Bus BLDC

GitHub Video

TL;DR: Custom manipulator with torque-controlled joints, real-time impedance control, and haptic feedback for teleoperation tasks.

A six degree-of-freedom robotic arm designed from the ground up with custom BLDC motor drivers and CAN-bus communication. Impedance control enables compliant interaction and haptic feedback for teleoperation, with all low-level firmware running on STM32 microcontrollers.
